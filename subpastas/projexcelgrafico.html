Resultados das Vendas

A tabela contém informações fictícias de uma loja de roupas. O objetivo aqui é mostrar, usando gráficos, como foi organizado as informações para identificar as melhores e piores vendas.

O gráfico de vendas foi criada usando o Gráfico de Casacata, que tem pequenos blocos que pode ter a sua posição alterada conforme os dados recebidos, neste caso, a ordem dos valores foi ordenado usando “Classificar e Filtrar”, em outra tabela que será falado em Análise de Dados, na página de projetos. O gráfico é composto por nomes dos vendedores há baixo, e no próprio gráfico, a quantidade de produtos vendidos por cada um deles(a), dessa forma, a visualização fica mais limpa e objetiva, com título que informa sobre o que está sendo mostrado, nomes dos responsáveis e quantidade de produtos.

O gráfico Lucro dos Vendedores é formado pelo gráfico de barras, contendo o total faturado por cada um deles(a), números formatados em moeda(real). Usando o mesmo critério, o gráfico é simples e objetivo com título e valores importantes, porém, um destaque para as cores, uma cor mais clara no fundo para destacar a cor mais escura que está responsável pela informação principal.

O gráfico Venda dos Produtos é formado pelo gráfico de barras, contendo o total faturado por cada item, números formatados em moeda(real). Usando o mesmo critério, o gráfico é simples e objetivo com título e valores importantes, porém, um destaque para as cores, uma cor mais clara no fundo para destacar a cor mais escura que está responsável pela informação principal.
